Instale as dependências:

```
npm install
```

Rode o projeto:

```
ng serve
```

Acesse no navegador: http://localhost:4200

🔧 Funcionalidades
Listar clientes

Cadastrar cliente

Editar cliente

📁 Estrutura básica
clientes/: componente com a lista de clientes

cliente-create/: formulário para criar e editar clientes

cliente.service.ts: comunicação com a API
